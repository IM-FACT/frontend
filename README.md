# IM.FACT Frontend

IM.FACT의 프론트엔드 애플리케이션입니다. Streamlit을 사용하여 구현되었습니다.

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 환경 변수 설정:
`.env` 파일을 생성하고 필요한 환경 변수를 설정합니다.

## 실행 방법

```bash
streamlit run app.py
```

## 주요 기능

- 환경 및 기후 관련 질문 입력
- 실시간 답변 생성
- 출처 정보 표시
- 사용자 친화적 인터페이스

## 기술 스택

- Streamlit
- Python
- FastAPI (백엔드 연동)
